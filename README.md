# Lógica de Programação em Linguagem C
Esses são alguns dos meus exercícios da matéria lógica de programação, foi utilizado a liguagem C para praticar funções, laço de repetição, condicionais, vetores e matrizes.

<h1>EXERCÍCIOS</h1>
<!--exercicio 1-->
<section>
  <h2>1. Calculadora modular com funções</h2>
  <p>Calculadora que retorna soma, subtração, multiplicação, divisão e resto de divisão por meio de funções, onde o usuário, por um switch, possa escolher as operações que quiser realizar até desejar sair.</p>
</section>

<!--exercicio 2-->
<section>
  <h2>2. Classificador de Números</h2>
  <p>Classificador de números com switch, que ao ler o número ele é classificado como par ou ímpar positivo, par ou ímpar negativo ou zero utilizando módulo e condicionais e retorna a classificação ao usuário.</p>
</section>

<!--exercicio 3-->
<section>
  <h2>3. Tabela de Operações com Vetor</h2>
  <p>Usuário digita 5 números, lidos por um vetor, para escolher se deseja somar todos, subtrair todos, multiplicar todos ou dividir o primeiro pelo último.</p>
</section>
